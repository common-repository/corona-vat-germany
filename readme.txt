Let WordPress and WooCommerce handle the upcoming 2020 tax rate changes automatically!


== Description ==

This is a flexible plugin that reduces the German VAT/MwSt. in WooCommerce from 19% to 16% (or from 7% to 5% for reduced rates) starting at July 1st until the end of the year.


== Frequently Asked Questions ==

= Does this plugin handle reduced VAT classes? =

Yes, you can select tax classes as normal and reduced VAT classes. They will be reduced to 16% or 5% accordingly.

= Will the taxes still be changed automatically after deactivating the plugin? =

No, please keep the plugin active as long as you wish to use it's function. The scheduled tasks will be removed on deactivation of the plugin.

= What is supported and what is not? =

Please let me know if you encounter any errors with this plugin.
Individual set-ups or adjustments to this plugin are not supported.


== Installation ==

After installation and activation of this plugin, make sure to visit the settings page and choose your tax classes. The automated tasks will be scheduled and rescheduled after each change.
